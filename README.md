# Simple Docker Compose for wordpress local development




##### Stack

- Wordpress 4.7.1
- Mysql 5.7
- Phpmyadmin (come on we all love him)
- Pureftp


##### Create containers

```
docker-compose up
```

##### Strating containers
```
docker-compose start
```

##### Stopping containers
```
docker-compose stop
```

##### Remove containers
To stop and remove all the containers use the **down** command
```
docker-compose down
```
